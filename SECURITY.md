# Sicherheitsrichtlinie

## Sicherheitslücken melden

Bitte veröffentliche mögliche Sicherheitslücken nicht als öffentliches GitHub-Issue. Sende stattdessen eine E-Mail an `maurice.keil1989@web.de` mit einer nachvollziehbaren Beschreibung, den betroffenen URLs und – falls möglich – Schritten zur Reproduktion.

Es werden keine Passwörter, Zugangsdaten, personenbezogenen Kundendaten oder funktionsfähigen Schadcodes angefordert. Bitte füge solche Informationen nicht in die Meldung ein.

## Unterstützte Version

Unterstützt wird ausschließlich der aktuell auf dem Branch `main` veröffentlichte Stand.
